🔬 Methodology

Missing Value Handling

KNN Imputation with k=5 neighbors. Estimates each missing value from the 5 most similar rows.

Feature Engineering

Three derived features: donation_density, recency_score, high_frequency_flag.

Algorithm

Random Forest Classifier — 100 trees, balanced class weight, 80/20 stratified split.

Validation

5-fold cross-validation for generalization, StandardScaler normalization.
